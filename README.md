#### CheatSheet

------

##### Mode IO
- 0 READ MODE
- 1 WRITE MODE

##### IO ADDRESS
- 0 TIME IO - LAST BIT IS TICK FLAG
- 1 BUTTON/SW IO - BUTTON in 0 and SW(11 - 4) in (8-1)
- 2 MIN-DEC SEGMENTS IO
- 3 MIN SEGMENTS IO
- 4 HR-DEC SEGMENTS IO
- 5 HR SEGMENTS IO
- 6 AM-PM

##### REG ADDRESS
- 0 TIME IO FLAG
- 1 SECOND REGISTER
- 2 MIN REGISTER
- 3 MIN-DEC REGISTER
- 4 HR REGISTER
- 5 HR-DEC REGISTER
- 6 BUTTON REG
